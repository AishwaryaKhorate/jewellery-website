# 💎 Jewellery Website

A responsive jewellery website built using **React** and **Vite**. The website showcases different jewellery collections with a clean and modern user interface.

## Features

- Home Page
- Image Slider
- Jewellery Categories
- Category Details
- Responsive Design

## Tech Stack

- React.js
- Vite
- JavaScript
- HTML
- CSS

## Project Structure

```
jewellery-website/
│
├── public/
├── src/
├── package.json
├── vite.config.js
└── README.md
```

## Run the Project

```bash
npm install
npm run dev
```
